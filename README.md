# Ehsan Moradi Portfolio

Static GitHub Pages site for Ehsan Moradi.

## Files
- `index.html`
- `style.css`
- `script.js`

## Deploy
Push these files to the root of the repository:
`ehsanwwe.github.io`

Then enable GitHub Pages:
Settings → Pages → Deploy from branch → `main` → `/ (root)`
